How to Use

1
Dashboard

메인 화면에서 전체 멤버 현황을 확인합니다. 팀/등록유형 필터로 원하는 그룹을 선택하세요. 팀 이름을 클릭하면 멤버 목록이 펼쳐집니다.

2
Individual Report

Dashboard에서 멤버 이름을 클릭하면 개인 리포트가 생성됩니다. 첫등록 멤버는 성장 리포트(4L 분석 포함), 재등록 멤버는 참여도 리포트(재등록 예측 지표 포함)가 출력됩니다.

3
Reports Tab

Reports 탭에서 전체 리포트 일괄 생성 또는 팀 현황 리포트를 만들 수 있습니다. 필터 설정 후 버튼을 클릭하세요.

4
PDF Export

Print / PDF 버튼을 클릭하면 브라우저의 인쇄 대화상자가 열립니다. "PDF로 저장"을 선택하면 A4 사이즈 PDF로 저장됩니다.

Data Update

출석 데이터 수정

data.js 파일에서 각 멤버의 attendance와 events 배열을 수정합니다.

  • true — 출석 (O)
  • false — 불참 (X)
  • null — 아직 미진행 (-)

Example

3회차까지 진행, 1회차 출석, 2회차 불참, 3회차 출석인 경우:

attendance: [true, false, true, null, null, null]

New Member / Team 추가

data.js의 teams 배열에 팀 객체를 추가하거나, 기존 팀의 members 배열에 멤버를 추가합니다.

Report Types

1. 참여도 리포트 (Participation Report)

재등록 멤버 대상. 팀 세션 출석률 + 이벤트 참석률 + 재등록 예측 지표를 제공합니다.

2. 성장 리포트 (Growth Report)

첫등록(뉴멤버) 대상. 출석 현황 + 관찰 포인트 + 팀 참여 제안 + 4L 분석 체크리스트를 포함합니다.

3. 팀 현황 리포트 (Team Overview)

전체 팀의 인원, 첫등록률, 회차별 출석 현황, 평균 출석률을 한 페이지에 비교합니다.

Metrics

소속감 지표

  • 첫등록률 — 시즌 내 신규 멤버 비율. 커뮤니티 유입 건강도를 나타냅니다.
  • 팀 세션 출석률 — 6회 세션 참석 비율. 팀 학습 몰입도의 핵심 지표입니다.
  • 이벤트 참석률 — PEST/AAR/HBR 참석. 팀 외 커뮤니티 확장 의지를 보여줍니다.
  • 재등록 전환율 — 시즌 종료 후 첫등록 멤버 중 다음 시즌 등록자 비율.

재등록 예측 기준

  • 팀 세션 출석률 80% 이상 → 재등록 가능성 높음
  • 이벤트 1회 이상 참석 → 커뮤니티 확장 관심
  • 시즌 후반(5-6회차) 연속 참석 → 소속감 유지 신호
